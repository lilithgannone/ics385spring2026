# Hawaii Island Cards: HW12a React Basics Assignment

## Description
This assignment involved creating a simple multi-component React application in order to practice core React concepts. Three Hawai'i island cards are rendered in this app. Each card displays an island name, a one-sentence description, and a visitor tip. The app is organized into three components: a main App.jsx file that holds the island data and renders the cards, an IslandCard.jsx that defines the card structure, and a Header.jsx that renders the page header. After watching the Udemy React course, I also decided to add very simple inline styling to the island name line. I also practiced using const CustomStyle. 

Simply, this app renders a header and three island cards with basic information about each island.

## Components Used/ Files Included
- App.jsx
- IslandCard.jsx
- Header.jsx

## Styling
I added a single line of inline styling to the header component to make the text blue and bolded. I also added a single line of inline styling to the island name in the IslandCard component to make it green.
